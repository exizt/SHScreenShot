# 기능
* 전체 캡쳐 : 스크린 전체의 영역을 캡쳐. 
* 영역 캡쳐 : 사각형 영역으로 캡쳐.



# 기능 구현 Details
공통 기능
* '설정되어 있는 폴더' 에 이미지 파일을 바로 생성한다. 


'전체 캡쳐' 기능
* 버튼을 누르거나 단축키를 누르면 스크린샷 이미지를 생성한다.
* 설정되어 있는 폴더에 자동으로 생성해준다.


'영역 캡쳐' 기능
* 사각형 영역으로 특정 영역을 선택해준다.
* 미리보기 이미지 생성
** 사이즈를 조절하거나, 위치를 옮길 때마다 '미리보기' 를 갱신해준다.
** 특정 시간 단위로도 '미리보기'를 갱신해준다. (구현중)
* 이미지 생성
** 단축키를 누르면 이미지를 생성해준다.
